Correct Answer: Number of classes in a continuous distribution

Struge's rule is used to find out the number of classes in a continuous distribution. It is a commonly used method for determining the appropriate number of classes for a histogram or frequency distribution. According to Struge's rule, the number of classes should be approximately equal to the square root of the total number of observations in the data set. The formula for Struge's rule is:

 

Number of classes = 1 + 3.322 log(n)

 

where n is the number of observations in the data set. The result of this formula should be rounded to the nearest whole number to obtain the final number of classes.

  • A.select appropriate class intervals
  • B.determine class intervals
  • C.determine class limits
  • D.determine midpoints of classes
Correct Answer: determine class limits

Steps for constructing a frequency distribution from a data set

1. If the number of classes is not given, decide on a number of classes to use.

2. Find the class width: Determine the range of the data and divide this by the number of classes. Round up to the next convenient number.

3. Find the class limits: You can use the minimum data entry as the lower limit of the first class. To get the lower limit of the next class, add the class width. Continue until you reach the last class. Then find the upper limits of each class.

4. Count the number of data entries for each class and record the number in the row of the table for that class.
