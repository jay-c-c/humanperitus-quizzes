The binary equivalent of is (-23)10is (2’ complement system for negative numbers is used) :-
  • A.111010010
  • B.111010001
  • C.111010111
  • D.111110001
Correct Answer: 111010001

Step 1: Let us Convert 23 into Binary

Step 1: 23/2, Quotient = 11 Remainder = 1 (Least Significant Bit)

Step 2: 11/2, Quotient = 5 Remainder = 1

Step 3: 5/2, Quotient = 2 Remainder = 1

Step 4: 2/2, Quotient = 1 Remainder = 0

Step 5: 1/2, Quotient = 0 Remainder = 1 (Most Significant Bit)

The Remainders, but in Reverse Order, will give you the Binary code. Start writing reminder from last step, towards first step.

In this example first we write 1 (Reminder of Step 5), then we write 0 (Reminder of Step 4), then we write 1 (Reminder of Step 3), then we write 1 (Reminder of Step 2), then we write 1 (Reminder of Step 1).

Finally, we have 10111, which is Binary equivalent of POSITIVE 23.

 

Step 2: Then the Positive binary number is represented on 4 or 8 bits - if needed, add extra 0s in front (to the left) of the base 2 number, up to the required length:

23(10) = 0001 0111 (we added three zeroes to the left)

 

Step 3: Now replace all the bits on 0 with 1s and all the bits set on 1 with 0s (reversing the digits):

(0001 0111) = 1110 1000

Step 4: To get the negative integer number representation on 8 bits, signed binary two's complement, add 1 to the number calculated above:

1110 1000 + 1 = 1110 1001

 

Hence, 111010001 is the Answer.
